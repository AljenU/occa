### Example: Inlined Kernels

We give an example with inlined OKL

### Compiling the Example

```bash
make
```

### Usage

```
> ./main --help

Usage: ./main [OPTIONS]

Example showing inlined OKL

Options:
  -d, --device     Device properties (default: "mode: 'Serial'")
  -h, --help       Print usage
  -v, --verbose    Compile kernels in verbose mode
```
